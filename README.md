# Scroll Pack Generator

A Vite + React prototype for opening 5e spell scroll packs with weighted rarity, a 75% conjuration / 25% staple split, and support for card-coded spell variants.

## Run locally

1. Install dependencies
2. Start the dev server

## Customize

Edit `src/data/spells.ts` to:

- add your full spell list
- adjust rarity weights
- add card-coded variants like `Summon Beast: Giant Ape`
- tune the pool split and pack size
